# Overview  
problem-forge는 대한민국 수능 스타일의 영어 문제를 다양한 유형으로 자동 생성해주는 에이전트 기반 프로그램입니다.  
주요 타겟은 수험생, 학원, 교사 등 입시와 관련된 넓은 사용자층입니다.  
영어 시험 준비를 위해 특정 지문을 기반으로 다양한 유형의 문제를 풀어보고 싶은 사용자에게, 대화형 에이전트를 통해 맞춤형 문제를 제공합니다.  
기존 서비스와 달리, 사용자의 입력(지문/문제)에 따라 실시간으로 다양한 유형의 문제를 생성하고, 채팅을 통해 추가 요청도 반영할 수 있다는 점이 차별점입니다.

# Core Features  
- 사용자는 문제를 PDF, 이미지, 텍스트 파일로 입력할 수 있습니다.  
- 기본적으로 5지 선다형 문제를 생성하며, 지원하는 문제 유형은 다음과 같습니다:
  - 글의 목적 파악
  - 글의 분위기 파악
  - 대의 파악(주장/요지/주제/제목 추론)
  - 함의 추론(특정 단어나 어구의 의미 추론)
  - 도표 문제(그래프/표 기반)
  - 내용 일치/불일치 파악
  - 어법 판단(밑줄 친 부분 중 어법상 틀린 것 찾기)
  - 어휘 적절성 판단
  - 빈칸 추론
  - 무관한 문장 찾기
  - 글의 순서 배열
  - 문장 넣기
  - 요약문 완성
  - 가리키는 대상이 다른 것 찾기
  - 인물의 심경 변화 찾기
- 문제 유형은 추후 추가될 수 있습니다.
- 생성된 문제와 해설, 정답은 항상 자동으로 제공되며, PDF 파일로 다운로드할 수 있습니다.
- 사용자는 채팅을 통해 추가 문제 유형을 요청할 수 있습니다.
- 문제 생성 난이도 조절 기능은 추후 확장 가능한 영역으로 남겨둡니다.

# User Experience  
- 주요 사용자는 고3 수험생, 영어 교사 등입니다.
- 사용 흐름:
  1. 웹페이지 접속
  2. 문제(지문) 업로드(PDF/이미지/텍스트)
  3. 에이전트가 다양한 유형의 문제 및 해설, 정답 생성
  4. 사용자가 채팅으로 추가 요청 시, 에이전트가 반영하여 문제 추가 생성
  5. 생성된 문제/해설/정답을 PDF로 다운로드
- 로그인 기능은 MVP에서는 제공하지 않습니다.

# Technical Architecture  
- google-adk를 활용하여 문제 변형, 해설 및 정답 생성, 세션 관리 등 에이전트 기능을 구현합니다.
- PDF/이미지 파일은 마크다운 또는 텍스트로 변환하여 에이전트에 전달하는 모듈이 필요합니다.
- 데이터는 별도로 저장하지 않습니다(비로그인, 무저장 구조).
- 외부 연동은 MVP에서는 최소화합니다.

# Development Roadmap  
## MVP
1. 사용자의 텍스트 입력 또는 파일(PDF, 이미지) 입력을 에이전트에게 전달하는 기능
2. 문제 또는 지문을 전달받아 다양한 유형의 문제를 생성하는 기능
3. 생성된 문제에 대한 해설 및 정답을 자동으로 제공하는 기능
4. 사용자가 생성된 문제, 해설, 정답을 PDF 파일로 다운로드할 수 있는 기능

## 추후 확장 방향
1. 사용자 로그인 및 개인화 기능
2. DB에 사용자 대화/생성 문제 저장
3. 문제 생성 난이도 조절 기능

# Logical Dependency Chain  
- PDF/이미지 파일을 에이전트에게 전달할 수 있는 구조가 가장 우선입니다.
- 최소 데모는 사용자가 문제를 입력하면 에이전트가 해당 문제를 이해했는지 답변하는 수준까지 구현하면 됩니다.

# Risks and Mitigations  
- **기술적 리스크:** PDF/이미지로 제공된 문제가 제대로 전달되지 않을 수 있음
- **해결 방안:** 이미 개발된 라이브러리(예: Tesseract, pdfplumber 등) 활용
